## Deploy

### GitHub Pages
1. Settings → Pages → Source: `main` → Save
2. Tu portfolio estará en `luispascualux.github.io/portfolio`

### Otros hostings
Sube la carpeta completa a Vercel, Netlify, o cualquier hosting estático.

## Desarrollo local

Abre `index.html` en tu navegador o usa un servidor local:

```bash
npx serve .
```
